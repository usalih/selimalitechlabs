# SALIMALITECH LABS - Professional Tech Hub Website

A modern, responsive one-page website for SALIMALITECH LABS, a world-class tech hub and workspace located in Gombe, Nigeria.

## 🎯 Overview

SALIMALITECH LABS is a tech hub designed to empower the Gombe community by providing access to world-class workspace, mentorship, and collaborative opportunities. This website showcases our services and mission to bridge the gap between local talent and global innovation.

- **Sophisticated Color Scheme**: Deep Forest (#013220), Emerald (#50C878), Neon Mint (#98FF98), White, and Charcoal

- **Glassmorphism Effects**: Elegant glass-style contact card with backdrop blur

## 🚀 Quick Start

### Option 1: Open Directly
1. Download or clone the repository
2. Open `index.html` in your web browser
3. No installation or dependencies required!

### Option 2: Serve Locally
If you have Python installed:
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```
Then visit `http://localhost:8000` in your browser.

Or if you have Node.js:
```bash
npx http-server
```

## 🔧 Customization Guide

### Change Contact Information
Find the Contact Section and update:
```html
<p>
    <i class="fas fa-map-marker-alt"></i>
    SALIMALITECH LABS, Gombe, Nigeria (NG)
</p>
```

### Modify Colors
Update the CSS variables in the `:root` selector:
```css
:root {
    --deep-forest: #013220;
    --emerald: #50C878;
    --neon-mint: #98FF98;
    --white: #FFFFFF;
    --charcoal: #1A1A1A;
}
```

### Update Services
Find the `services-grid` section and edit the service card HTML:
```html
<div class="service-card">
    <i class="fas fa-[icon-name] service-icon"></i>
    <h3>Service Name</h3>
    <p>Service description...</p>
</div>
```
